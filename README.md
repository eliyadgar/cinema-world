# cinema-world
Dynamic Yield home assignment 

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

#### Added Features:
1. Added favorites list where a user can add/remove movies to his wish list in the specific session
2. Marked unavailable showtimes as disabled
3. Prevent multiple http calls with the same search term within 1 hour
